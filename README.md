## Projects

### 1. Stewart Platform — Thesis Project (2020)
**6-DOF Parallel Robot Inverse Kinematics Analysis**

- Inverse kinematics calculation for 6x6 Stewart Platform
- Simulation in V-REP robot simulation environment
- Advisor: Assoc. Prof. Dr. Ali SAYGIN

📁 Full project: [Stewart_Platform repo](https://github.com/Mehmet-Haydar/Stewart_Platform)

---

### 2. PLC Sun Position Alert System (2019)
**TIA Portal application generating alerts based on sun position**

- Calculates sun position from time and GPS coordinates
- Generates warnings for solar-dependent operations
- Implemented in Siemens TIA Portal
- Advisor: Assoc. Prof. Dr. Ali SAYGIN

📁 Folder: `plc-sun-position-alert/`

---

### 3. Autonomous Solar Tracking System (2018)
**Microcontroller-based solar panel positioning**

- Uses internal RTC + GPS for position calculation
- No external light sensors required
- Developed in CCS C compiler
- Advisor: Assoc. Prof. Dr. Mehmet DEMİRTAŞ

📁 Folder: `solar-tracking-system/`
